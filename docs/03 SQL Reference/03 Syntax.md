## GROUP BY

**Limitations**

Can `GROUP BY` functions in the `SELECT` clause but not by aliases.

Can't `GROUP BY` the result of the `CAST` function, can can `GROUP BY` individual type cast functions like `NUMERIC`, `VARCHAR` and `BOOLEAN`.
