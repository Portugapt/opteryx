## WHERE

`_PARTITION` is a special term, allowing you to specify partitions in SQL, can be
used in WHERE clauses like any other field.

`WHERE _PARTITION BETWEEN <start_date> AND <end_date>`
`WHERE _PARTITION > "2021-01-02"`
