## Comparisons

Operator     | Description              | Example | Result
------------ | ------------------------ | ------- | ------
`<`          | less than                | 2 < 3   | TRUE
`>`          | greater than             | 2 > 3   | FALSE
`<=`         | less than or equal to    | 2 <= 3  | TRUE
`>=`         | greater than or equal to | 4 >= 8  | TRUE
`=`          | equal to                 | 3 = 4   |	FALSE
`<>`         | not equal to             | 2 <> 2  | FALSE
`LIKE`       | pattern match            |         |
`~`          | regular expression match |         |
